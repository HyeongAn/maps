# maps

monorepo로 이루어진 maps 프로젝트 입니다. 각각의 버전은 아래와 같습니다.

```
NodeJS v18.19.0
NextJS v14.2.13
React v18^
yarn v4.4.1
```

apps 하위에 nextJS로 이루어진 web버전이 있습니다. (native 버전은 apps하위에 추가될 예정입니다.)
실행방법은 아래와 같습니다.

```
yarn workspace next-app run dev
```
